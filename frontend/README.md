# Netflix Clone - Frontend

Responsive Netflix-style React frontend for the internship assignment.

## Features

- Responsive desktop and mobile UI
- Netflix-like home page
- Hero banner section
- Movie/series rows
- Navbar with NETFLIX logo, menu items, search, and mobile menu icon
- Search functionality using backend movie data
- React Router pages: `/login`, `/browse`, `/movies/:id`
- Loading skeleton rows
- Dark Netflix-style UI

## Setup

```bash
npm install
npm run dev
```

Frontend runs on:

```txt
http://localhost:5173
```

## Environment

Create `.env` if your backend URL is different:

```env
VITE_API_BASE_URL=http://localhost:5000
```

## Demo Login

Use any valid email and a password with minimum 4 characters.

Example:

```txt

Password: 123456
```
